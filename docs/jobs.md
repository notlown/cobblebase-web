# Jobs

Cobblebase features **44 unique jobs** across 10 categories. Every Pokemon species has hand-crafted skill assignments -- not based on type alone, but on the identity and lore of each species.

---

## Gathering (6 jobs)

Pokemon that harvest, fish, mine, and collect resources from the world around your base.

### Harvester

Picks apricorns, crops, berries, mints, tumblestones, and nether wart from nearby farmland and plants.

| Property | Value |
|----------|-------|
| Base Cooldown | 10 seconds |
| Search Radius | 20 blocks |
| Example Pokemon | Venusaur, Scizor, Celebi |

### Fishing

Navigates to nearby water and catches fish and loot using Minecraft's fishing loot table.

| Property | Value |
|----------|-------|
| Base Cooldown | 60 seconds |
| Search Radius | 10 blocks |
| Requires | Water nearby |
| Example Pokemon | Gyarados, Blastoise, Kyogre |

### Mining

Digs for ores, fossils, gems, and tumblestones. Loot is divided into four tiers:

- **Common:** Tumblestone, raw copper, raw iron
- **Uncommon:** Raw gold, lapis lazuli, type gems
- **Rare:** Diamond, fossils
- **Ultra Rare:** Ancient debris, netherite scrap

| Property | Value |
|----------|-------|
| Base Cooldown | 300 seconds (5 min) |
| Search Radius | 10 blocks |
| Example Pokemon | Steelix, Excadrill, Ting-Lu |

### Honey Collect

Produces honey bottles, honeycomb, and honey blocks. No beehive needed -- the Pokemon produces honey on its own.

| Property | Value |
|----------|-------|
| Base Cooldown | 120 seconds |
| Search Radius | 10 blocks |
| Example Pokemon | Vespiquen, Beedrill, Combee |

### Archeologist

Excavates ancient relics and artifacts including prismarine, nautilus shells, and other treasures.

| Property | Value |
|----------|-------|
| Base Cooldown | 80 seconds |
| Search Radius | 10 blocks |
| Example Pokemon | Nidoking, Steelix, Nidoqueen |

### Diving

Dives underwater to find treasures and rare items. Pokemon must be near water to dive. Loot includes Cobblemon items like Deep Sea Scales, Prism Scales, and fossils, plus vanilla treasures like Heart of the Sea.

- **Common:** Relic Coin, Water Gem, Prismarine Shard, Dive Ball, Net Ball
- **Uncommon:** Deep Sea Scale, Deep Sea Tooth, Mystic Water, Lure Ball
- **Rare:** Prism Scale, Shell Bell, Water Stone, Dragon Scale, Nautilus Shell, Fossils
- **Ultra Rare:** Heart of the Sea

| Property | Value |
|----------|-------|
| Base Cooldown | 360 seconds (6 min) |
| Search Radius | 15 blocks |
| Requires | Water nearby |
| Example Pokemon | Lapras, Kyogre, Manaphy |

## Support (2 jobs)

### Healer

Directly heals Pokemon in the pasture by a percentage of their max HP. Also revives fainted Pokemon. Prioritizes the Pokemon with the lowest HP percentage.

| Property | Value |
|----------|-------|
| Base Cooldown | Continuous (0s) |
| Search Radius | 10 blocks |
| Healing Range | 5-25% max HP per tick (scales with proficiency) |
| Example Pokemon | Blissey, Xerneas, Zacian |

### Mentor

Lets every Pokemon in the same pasture gain passive XP over time. **Without an active Mentor, Pokemon do not gain XP from working at all** — the Mentor is the source of passive XP, not a multiplier on top of base gains. The highest proficiency Mentor wins; multiple Mentors do not stack.

| Property | Value |
|----------|-------|
| Base Cooldown | Continuous (passive) |
| Search Radius | 10 blocks |
| Prof 1 Boost | +20% XP |
| Prof 5 Boost | +100% XP (double) |
| Example Pokemon | Alakazam, Latios, Mesprit |

The max boost is configurable via Cloth Config (default: 100%).

## Exploration (1 job)

### Scout

Explores the surrounding area to discover wild Pokemon, structures, and biomes. Discoveries are permanent and visible in the Discovery GUI tab.

| Proficiency | Range | Discovers |
|-------------|-------|-----------|
| 1-2 | 50-100 blocks | Wild Pokemon only |
| 3 | 120 blocks | + Structures (Villages, Mineshafts, Shipwrecks) |
| 4-5 | 160-200 blocks | + Rare Biomes (Mushroom Fields, Deep Dark, Cherry Grove) |

| Property | Value |
|----------|-------|
| Base Cooldown | 300 seconds (5 min) |
| Base Search Radius | 100 blocks |
| Example Pokemon | Ninjask, Spectrier, Rayquaza |

Wild Pokemon sightings auto-expire after 30 minutes. Structures and biomes persist permanently.

## Recruiting (2 jobs)

Pokemon that attract and spawn new wild Pokemon near your base.

### Friend Recruiter

Spawns wild Pokemon of the same type near your base. Uses official Cobblemon 1.7.3 spawn bucket data for accurate rarity distribution.

| Property | Value |
|----------|-------|
| Base Cooldown | 300 seconds (5 min) |
| Search Radius | 15 blocks |
| Example Pokemon | Togekiss, Sylveon, Gardevoir |

Rarity distribution scales with proficiency:

| Proficiency | Common | Uncommon | Rare | Ultra-Rare |
|-------------|--------|----------|------|------------|
| 1 | 93.8% | 5.0% | 1.0% | 0.2% |
| 3 | 90.3% | 7.5% | 1.5% | 0.3% |
| 5 | 86.8% | 10.0% | 2.0% | 0.4% |

### Legendary Recruiter

Spawns rare and legendary Pokemon near your base. Much longer cooldown than Friend Recruiter.

| Property | Value |
|----------|-------|
| Base Cooldown | 600 seconds (10 min) |
| Search Radius | 20 blocks |
| Example Pokemon | Arceus, Mew, Jirachi |
